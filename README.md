# Apoio para executar o projeto:

-  **Instalar yarn:**
```
  npm install --global yarn
```
<br>

- **Verificar versão do yarn:** 
```
  yarn --version
```
<br>

- **Instalar thon-ui:**
```
   yarn add thon-ui
```

- **Instalar date-fns:**
```
  yarn add date-fns
```
<br>

- **Instalar classname:**
```
  yarn add classnames  
```
<br>

- **Descompactar o Arquivo `.env` ques está zipado**

## Iniciar o projeto: 
```
  yarn dev
```
